Sobre Mim

Olá, me chamo Heclair sou formado em Engenharia de Produção pela ETEP Faculdades, possuo MBA em Logistica e Supply Chain pela Universidade Anhembi Morumbi e devido a minha constante atualização estou me formando em Desenvolvimento de Software Multiplataforma pela ETEC Professor Francisco Moura.

Venho atuando na área de Compras desde 2016 tendo atuado em multinacionais de grande porte como CAOA Chery, Bayer, Cielo e Ball, experiências focadas em redução de custo através da metodologia strategic sourcing nas categorias de RH, Facilities e Tecnologia. Atualmente possuo um foco na implementação de processos e harmonização de dados Globais entre América do Norte e Europa desenvolvendo sistemas low code com Power Apps e gerando insights com Power BI e Qlik Sense.

Treinamento Metodologia Ágil

O projeto tinha como objetivo a criação de um site utilizando HTML, CSS e o framework BootStrap para a apresentação de um treinamento de metodologia ágil para os colaboradores da Fatec Jacareí.

Neste projeto houve a oportunidade de obter o primeiro contato com a metodologia ágil de projeto, utilizando as ferramentas Kanban e Scrum, além de aprender a utilizar o framework BootStrap e ser capaz de gerenciar a equipe na posição de Scrum Master.

Bike Pass

Com o objetivo de atender o nosso Stakeholder Gen App o projeto consistia no desenvolvimento de uma plataforma onde usuários podem disponibilizar suas bicicletas para locação

O projeto foi desenvolvido utilizando o framework React para o frontend e typescript com node e banco de dados Postgree SQL para o backend. Neste projeto houve a oportunidade de aprofundar o conhecimento em controllers, persistência de banco de dados, implementação de CRUD e navegação.

Satélite

O projeto Satélite tinha como objetivo atender as necessidades da empresa Visiona que foi o stakeholder do projeto. O objetivo primário deste desenvolvimento era prover um sistema de gestão de produtividade das atividades dos colaboradores da Visiona, oferecendo os principais KPIs solicitados pelo stakeholder

Neste desenvolvimento houve um foco no uso das tecnologias de React com typescript e o primeiro contato da equipe com banco de dados não relacional, onde o banco de dados escolhido foi MongoDB e para exibição dos KPIs alternamos entre Power BI e Material UI.

NutriPerry

Com o objetivo de desenvolver uma aplicação móvel para a área da saúde onde o usuário é capaz de fazer a gestão do seu consumo calórico a equipe The Perry Dev desenvolveu o aplicativo móvel "NutryPerry"

Tive a oportunidade de aprender a utilizar o framework React Native para o desenvolvimento do aplicativo e assumi a frente de desenvolvimento de backend incluindo a arquitetura de dados, onde optamos pelo uso de um banco de dados não relacional para armazenar as informações dos usuários, além de fazer uso de uma API para realizar a autenticação de login do usuário e consulta de tabela nutricional de alimentos utilizando o Open Food Facts.
